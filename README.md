# Quilt Developer Wiki

Installed dependencies:

```bash
pnpm install
```

Run dev server with:

```bash
pnpm run dev

# or start the server and open the app in a new browser tab
pnpm run dev -- --open
```

## Building

To create a production version of your app:

```bash
pnpm run build
```

You can preview the production build with `pnpm run preview`.
